Generative UI

3 types of gen ui
- Controlled gen ui: where gen ui is already coded in
- Declarative gen ui: react + zod; declare the components: by creating a catalog- desc, when to use by agents, what data will it injected with (a2ui by google): long tail of user facing services + internal uses
- Open gen ui: writing JavaScript (open gen ui) + MCP apps (embed excalidraw or mermaid diagram gen inside the chat)

future of agentic ui
- enablemet(stack to assemble ui)
- insights(new analytics or insights to track)


- CLUF (continuous learning from user feedback)

The Three Components of an Agent
An agent is not just a model. It is a model inside a system with three distinct parts.

1. Model
The reasoning and language engine: decides what to do next.
- Gemma 4 (Goo[gle])
- Qwen 3.6 (Ali[baba])
- Kimi K2.6 (M[oonshot] AI)
- MiniMax M[...] Max)
- GPT-OSS (Op[enAI])

2. Runtime
Manages the loop, context, tool calls, retries, and state.
- Prompt assembly
- Tool execution
- Guardrails and recovery
- Integration layer

3. Tools
The external capabiliti[es an] agent can call.
- MCP Servers
- APIs and CLIs
- Code execution
- Custom tools
